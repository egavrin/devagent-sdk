# Security Policy

## Reporting a Vulnerability

Please do not open public GitHub issues for security vulnerabilities.

Use GitHub private vulnerability reporting for this repository if it is enabled. If it is not
available, contact the maintainer privately before disclosing details publicly.

## Supported scope

This repository covers:

- protocol types
- schema validation behavior
- fixture and compatibility integrity

## Out of scope

- provider-side model behavior
- local machine misconfiguration
- experimental executor adapters that do not rely on this repo's validated path
